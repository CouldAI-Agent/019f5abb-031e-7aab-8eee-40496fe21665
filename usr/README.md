# Plastic Reduction Project Report

An interactive dashboard application for tracking and reporting on plastic usage reduction initiatives.

## Features
- **Project Overview**: A high-level summary dashboard with key metrics (Total Reduced, Target Completion, Active Participants) and recent milestones.
- **Statistics**: Visual progress bars showing the steady decline in monthly plastic waste over time.
- **Action Plan & Roadmap**: A clear roadmap of completed, in-progress, and planned initiatives to eliminate single-use plastics.
- **Cross-Platform Responsive**: Fully responsive UI built with Flutter, adapting seamlessly between mobile, tablet, and web layouts using a bottom navigation bar on mobile and a side navigation rail on desktop.

## Tech Stack
- **Framework**: Flutter
- **Language**: Dart
- **Design System**: Material Design 3

## Setup Instructions
1. Ensure Flutter is installed on your machine.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the application.
